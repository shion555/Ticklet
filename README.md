# Ticklet

Ticklet is a macOS task management app built with SwiftUI and SwiftData.

## Current Features

- Create, edit, and complete tasks
- Organize tasks with lists
- Group tasks by date
- Manage recurring tasks
- Filter and review completed tasks

## Requirements

- macOS
- Xcode 16 or later

## Run Locally

1. Open `Ticklet.xcodeproj` in Xcode.
2. Select the `Ticklet` scheme.
3. Run the app on macOS.

## Run Tests

Run tests from Xcode or from the command line:

```sh
xcodebuild test -project Ticklet.xcodeproj -scheme Ticklet -destination 'platform=macOS'
```

## Development Status

Ticklet is under active development. The repository currently focuses on the core macOS app, tests, and the minimum project setup needed for open source collaboration.
